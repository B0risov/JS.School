# React ToDo List

### Run project

Install all node modules:
```shell
npm i
```

Start App with Local DB:
```shell
npm run start
```

Start App **without** Local DB:
```shell
npm run serve
```

Build project:
```shell
npm run build
```

**Description:** *ToDo List using React with Redux and Router.*
